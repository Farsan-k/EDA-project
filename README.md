# EDA-project

Project Overview

This project analyzes job market data to understand salary trends, industry demand, role-based compensation, and geographic salary differences using EDA and statistical testing.

Instructions to Run

Clone the repository

Install dependencies: pandas, numpy, matplotlib, seaborn, scipy

Run notebooks in sequential order

View figures in /reports/figures/

Key Insights

IT roles dominate the job market and offer higher compensation.

Salary is influenced by experience, role, industry, and location.

Metro cities and senior technical roles provide the highest salary potential.
